# Sentinel Autogard

Sentinel Autogard is an AI-driven market, project, tender, and site intelligence platform built around AUTOGARD.

## Initial Market

Switzerland is the first pilot market for opportunity discovery, project intelligence, tender monitoring, and site analysis.

## Core Modules

- Company Twin
- Market Intelligence
- Project Radar
- Tender Radar
- Opportunity Intelligence
- Partner Intelligence
- Competitor Intelligence
- Site Audit
- AI Sales Copilot

## Site Audit 1.0

The first implementation milestone focuses on a CesiumJS-based geospatial workspace for:

- locating sites by address or coordinates;
- visualizing terrain, satellite imagery, and 3D geospatial context;
- defining and editing site perimeters;
- measuring distance, perimeter, and area;
- creating structured Site records;
- preparing future integrations with street-level imagery, AI vision, security layers, and AUTOGARD product placement.

## Technology Direction

- Next.js
- CesiumJS
- PostgreSQL / PostGIS
- Object storage for documents and imagery
- Vector search for knowledge retrieval
- Knowledge graph where relationship analysis adds value

## Language Policy

All source code, UI text, schemas, API definitions, technical documentation, repository content, and development artifacts are written in English.

## Status

Early architecture and Site Audit MVP development.
